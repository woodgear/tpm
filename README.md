# tpm
tpm: template package manager
# how to use
## install
```rust
cargo install --git https://github.com/woodgear/tpm
```
## add you template repo
```
tpm add https://github.com/woodgear/t
```
## list all template
```
tpm list
```
## create a instance from template
```
tpm new $name
```
## update you repo
```
tpm update
```
